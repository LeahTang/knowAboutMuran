Polygons Image Learning

This project is to use evolution algorithm to do the image machine learning. One hundred polygons will be randomly generated at the beginning of the program. For each polygons, they might have different colors, size, level of transparency and position on a canvas. For each generated images, they will be compared with target image, and caluculated accuracy. With the programming running, the population (image) will evolute,more like the target iamge, becasue the evolution algorithm will select the better image to generate next generation. This kind of repetitive processes is called polygon image learning (complex vectors machine learning).

What I did in this project is to impove existing evolution algorithm in DEAP library to make more accurate image with less generations (prepitity times). To do that, I used data analysis and virtualization tools to help me to estimate whether the parameters is the best or not, or whether my algorithm is improved.

Read my research and report of doing this project.
